# Global AI Context

This folder contains shared instructions, process notes, prompts, and coding rules for AI agents working in this project.

## Files

- `agent-task-flow.md`: mindset and file-creation flow for each AI task.
- `coding-rules.md`: shared coding rules.
- `prompt-templates.md`: reusable prompt templates.

## Usage

Before starting a task, read the affected module docs first. Use `agent-task-flow.md` to decide whether the task needs a plan, review, changelog, decision, or rule update.
