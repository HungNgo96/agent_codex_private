# Frontend

This repository does not include a production frontend. Frontend guidance is reserved for projects that copy this template into a UI application.

When frontend work is added:

- Document the design system in `references/design-system-reference-llms.txt`.
- Keep user-facing flows in `product-specs/`.
- Add component or page architecture notes under `design-docs/`.
- Add verification expectations to `QUALITY_SCORE.md`.
