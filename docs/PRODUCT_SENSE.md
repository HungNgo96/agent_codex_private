# Product Sense

The product is a reusable agent-team template for software projects.

## Target Users

- Developers using Codex, Cursor, or Claude Code.
- Leads coordinating bounded agent work.
- Reviewers who need clear evidence before accepting changes.

## Product Direction

- Make agent delegation predictable.
- Keep context in the repository.
- Provide local verification before completion claims.
- Avoid custom runtimes until the manual workflow is proven.
