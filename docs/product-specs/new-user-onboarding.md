# New User Onboarding

## Goal

Help a developer copy this template into a project and run a disciplined lead/worker agent workflow with verification.

## Flow

1. Read `AGENTS.md`.
2. Read `README.md`.
3. Choose a relevant workflow from `workflows/`.
4. Create a task brief from `templates/task-brief.md`.
5. Dispatch bounded workers only when parallel work is useful.
6. Collect handoffs using `templates/handoff-note.md`.
7. Run the local harness before reporting completion.

## Success Criteria

- The user can identify the lead and worker responsibilities.
- The user can find the right prompt and workflow for the task.
- The user can run the harness and inspect the generated report.
